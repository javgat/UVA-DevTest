Crear Pregunta

Una vez crees la pregunta, se te redirigirá a el panel de edición de pregunta para que configures las opciones de respuesta correspondientes al tipo seleccionado

Las opciones para responder a la pregunta, y la información respecto a cual es la respuesta esperada, la podrás introducir en la siguiente página.
También podrás modificar todos los datos de la pregunta, incluso el tipo.
